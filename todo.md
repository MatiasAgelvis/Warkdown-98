# Warkdown 98 - Ideas & TODO

## Feature Ideas

### Image Upload & Desktop Icon Integration
Users should be able to upload images directly into the application. Once uploaded, images appear as draggable icons on the desktop (similar to the 98-inspired UI). These image icons can be:
- Dragged around the desktop interface
- Linked within the markdown editor as image references
- Inserted directly into markdown by dragging them into the editor content area

This would create a more immersive experience where the application manages a collection of media assets that users can easily reference in their markdown documents, staying true to the retro Windows 98 aesthetic.
